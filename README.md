# 🛒 Supermarket Sales Dashboard

## 📌 Project Overview
The Supermarket Sales Dashboard is an interactive data analysis project created using Microsoft Excel. This dashboard helps analyze supermarket sales performance across different cities, product categories, customer types, and time periods.

The dashboard provides visual insights into sales trends, customer behavior, product ratings, and city-wise performance to support business decision-making.

---

## 🎯 Objectives
- Analyze supermarket sales performance
- Compare sales across different cities
- Understand customer purchasing behavior
- Evaluate product ratings
- Identify top-performing product categories

---

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

---

## 📂 Dataset Information
The dataset contains supermarket transaction records including:

- Branch
- Customer Type
- Gender
- Product Line
- Sales
- Rating
- Month
- City

---

## 📊 Dashboard Features

### 1. Sales Analysis
- Compare sales across cities
- Identify high-performing branches
- Track sales distribution

### 2. Rating Analysis
- Product-wise customer ratings
- Compare customer satisfaction

### 3. Product Sales Per City
- Analyze product performance by location
- Compare city-wise demand

### 4. Interactive Filters (Slicers)
Users can filter dashboard using:
- Branch
- Customer Type
- Gender
- Month

---

## 📈 Key Insights
- Compare sales across Mandalay, Naypyitaw, and Yangon
- Analyze customer preferences
- Evaluate product performance
- Monitor rating distribution
